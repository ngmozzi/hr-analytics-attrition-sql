## Dataset

This project uses the IBM HR Analytics Employee Attrition and Performance dataset.

The dataset is not included in this repository. It can be downloaded for free from Kaggle:

[IBM HR Analytics Attrition Dataset on Kaggle](https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset)

Once downloaded, the CSV file should be named:
`WA_Fn-UseC_-HR-Employee-Attrition.csv`
